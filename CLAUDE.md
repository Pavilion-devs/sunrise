# Cluster 4 — Migrations ⚠️ TACKLE LAST

## Mission
Help projects migrate to Solana from other ecosystems (EVM, etc.) using
Sunrise. Build bridges, migration tooling, or onboarding guides.

## Bounties This Cluster Targets

| Bounty | Sponsor | Track |
|--------|---------|-------|
| Migrations | Sunrise | Bridges, migration tooling, onboarding guides |

## Core Concept
- Sunrise as the bridge/migration layer
- Target: projects wanting to move from EVM chains to Solana
- Could be tooling (scripts, CLIs), a guide platform, or a bridge UI
